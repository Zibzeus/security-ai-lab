# Lab Exercise SOP

1. Create or update the active engagement scope.
2. Confirm time synchronization on agent, BAS, targets, ExtraHop, and CrowdStrike.
3. Record baseline health and telemetry.
4. Select one skill and one measurable hypothesis.
5. Review the generated plan and required approvals.
6. Execute; preserve command, operation ID, timestamps, and raw evidence.
7. Correlate endpoint and network telemetry.
8. Apply cleanup and verify target state.
9. Complete the report and add durable lessons to the knowledge directory.
