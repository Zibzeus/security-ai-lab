# Alert Triage Baseline

1. Confirm alert time, affected asset, user, and sensor.
2. Separate observed telemetry from enrichment and analyst assumptions.
3. Establish a timeline and search for corroborating evidence from another source.
4. Map behavior to ATT&CK only when the evidence supports the technique.
5. Record confidence and evidence gaps.
6. Containment requires human approval and a working, scoped connector.
